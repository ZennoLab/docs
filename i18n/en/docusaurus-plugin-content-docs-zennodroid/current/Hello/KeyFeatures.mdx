## Main Features  
### Repeating Actions  
Easily record and replay actions in Android apps.  

- ![Full Android Automation](./assets/ZennoDroid/ZennoDroid_pic2.png) **Full Android Automation**    
Fill out forms, collect data, explore apps in a dedicated device window.  
- ![Multithreading ](./assets/ZennoDroid/ZennoDroid_pic3.png) **Multithreading**   
Run projects in multiple threads at once (work with several Android devices in parallel).  
- ![Bot Interface](./assets/ZennoDroid/ZennoDroid_pic4.png) **Bot Interface**   
Design beautiful and user-friendly interfaces for your bots.  
_______________________________________________
### Form Filling  
Automatically fill out forms with the data you need.  

- ![Scheduled Launch](./assets/ZennoDroid/ZennoDroid_pic5.png) **Scheduled Launch**    
Set your tasks to run exactly when you want.  
- ![User Data Generation](./assets/ZennoDroid/ZennoDroid_pic6.png) **User Data Generation**   
Automatically generate user data—from names to addresses.  
- ![Proxy Support](./assets/ZennoDroid/ZennoDroid_pic7.png) **Proxy Support**   
You can set up proxies for your Android virtual machine.
_______________________________________________
### Button Clicking  
Automate button and link presses.  

- ![User Emulation System](./assets/ZennoDroid/ZennoDroid_pic8.png) **User Emulation System**   
Interact with the screen using the keyboard: Tap, Long Tap, and Swipe. Actions will look just like a real person is doing them.  
- ![Virtual Machine Management](./assets/ZennoDroid/ZennoDroid_pic9.png) **Virtual Machine Management**   
From creating an Android virtual machine to fully resetting the device.
_______________________________________________
### Data Collection  
Extract data from any apps.  

- ![App Structure Analysis](./assets/ZennoDroid/ZennoDroid_pic10.png) **App Structure Analysis**   
Use tools to analyze the XML code of apps.  
- ![Parsing Tools](./assets/ZennoDroid/ZennoDroid_pic11.png) **Parsing Tools**   
Collect, process, and save data in whatever format works for you. 
- ![Mail](./assets/ZennoDroid/ZennoDroid_pic12.png) **Mail**   
Work with mailboxes: search, analyze, and pull info from emails.
_______________________________________________
### Device Emulation  
Emulate any device and its settings: model, IMEI, and more.  

- ![Parameter Emulation](./assets/ZennoDroid/ZennoDroid_pic13.png) **Parameter Emulation**   
Emulate any Android device settings and fingerprints—from device model to mobile operator.    
- ![Superuser Mode (root)](./assets/ZennoDroid/ZennoDroid_pic14.png) **Superuser Mode (root)**   
Get unlimited rights on an emulated device by enabling root access.  
- ![Working with Apps](./assets/ZennoDroid/ZennoDroid_pic15.png) **Working with Apps**   
Install apps from APKs, get the app list, and remove them as needed.
_______________________________________________
### Working with Any Data  
Work with text, tables, images, databases, and other data.  

- ![Data Management](./assets/ZennoDroid/ZennoDroid_pic16.png) **Data Management**   
Interact with any type of data: from text and images to Google Sheets with databases.  
- ![HTTP and FTP Support](./assets/ZennoDroid/ZennoDroid_pic17.png) **HTTP and FTP Support**   
Make your projects faster with HTTP and FTP requests.  
- ![Integration with 35 Services](./assets/ZennoDroid/ZennoDroid_pic18.png) **Integration with 35 Services**   
In one click, connect captcha-solvers, SMS activation, content creation, proxy services, and more.
_______________________________________________
### Extra Features  
- ![Your Own Code](./assets/ZennoDroid/ZennoDroid_pic19.png) **Your Own Code**   
Expand the program’s abilities using C#, JavaScript, and the command line.  
- ![Logic Operations](./assets/ZennoDroid/ZennoDroid_pic20.png) **Logic Operations**   
Set up different actions based on conditions.  
- ![Proxy Checker](./assets/ZennoDroid/ZennoDroid_pic21.png) **Proxy Checker**   
Check proxies for functionality, speed, anonymity, country, type, and 10+ more parameters.
