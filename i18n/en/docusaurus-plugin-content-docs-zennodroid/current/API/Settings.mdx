## Overview  
**`IDroidSettingsAPI Settings`**  

This object is for managing device settings. With it, you can set basic parameters: language, time, geolocation, and other system settings. It also provides tools for working with [LSPosed in ZDE](../Enterprise/LSPosed), making it easy to spoof phone data.

It mirrors the logic of the **Device Settings** cube for [**Lite/Pro**](../Android/ProLite/setting) and [**Enterprise**](../Android/Enterprise/setting) versions.  
_______________________________________________
## Methods  
### Set device language
```csharp
void SetLanguage(string language)
```

*Parameters:*  
```csharp
string language // Specify the language (for example, RU).
```

#### Example  
```csharp
var settings = instance.DroidInstance.Settings;

settings.SetLanguage("RU"); // Set the device language
```
_______________________________________________
### Set device time
```csharp
void SetTime(string dateTime)
```

*Parameters:*  
```csharp
string dateTime // Time as a string (for example, "12.05.2025 14:30").
```

*Overload:*  
```csharp
DateTime dateTime // Time as a DateTime object.
```

#### Example  
```csharp
var settings = instance.DroidInstance.Settings;

settings.SetTime(DateTime.Now); // Set the current time
```

_______________________________________________
### Generate JSON for random data spoofing
```csharp
string GenerateRandomLSPosedSettings(string countryName) 
```

*Parameters:*  
```csharp
string countryName // Country name for spoofing.
```

*Overload:*  
```csharp
IDeviceFilter filter // Filter for generating specific spoofed data.
```

#### Example  
```csharp
var settings = instance.DroidInstance.Settings;

var country = project.Profile.Country; // Country for generation
var json = settings.GenerateRandomLSPosedSettings(country); // Get JSON for LSPosed spoofing
```
_______________________________________________
### Apply spoofed phone data from JSON
```csharp
void SetLSPosedSettings(string json) 
```

*Parameters:*  
```csharp
string json // The JSON value for LSPosed spoofing.
```

#### Example  
```csharp
var settings = instance.DroidInstance.Settings;

var country = project.Profile.Country; // Country for generation
var json = settings.GenerateRandomLSPosedSettings(country); // Get JSON for LSPosed spoofing
settings.SetLSPosedSettings(json); // Apply device data spoofing
```
