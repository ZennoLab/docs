## Описание.  
**`IDroidHierarchyAPI Hierarchy`** 

Предоставляет доступ к дереву элементов интерфейса текущего экрана устройства. Используется для получения структуры UI, которая представляет собой XML-дерево всех элементов и их взаимосвязей. 
_______________________________________________
## Методы
### Получить XML древо текущего экрана.
```csharp
string GetLayout()
```   

#### Пример.  
```csharp
var xml = instance.DroidInstance.Hierarchy.GetLayout(); // Получить XML текущего экрана устройства
```
