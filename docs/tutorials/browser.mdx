## Начало работы с браузером

Покажем, как быстро начать работать с браузером.  

С **ZennoDroid** вы можете автоматизировать любые действия в мобильном браузере.  
В этой статье помимо текстового описания стандартных операций с браузером вы также найдёте видеогайды и шаблон с настройкой.  

### Стандартные действия в браузере.

> ***Краткое содержание**:*   
> *Открываем браузер на устройстве → вводим в поисковую строку запрос «ZennoLab» → переходим по ссылке на страницу ZennoDroid → возвращаемся на главный экран.*  

#### Подробная инструкция:  

1. Через **Установленные приложения** открываем браузер (в нашем случае это Chrome).  
> Действие *«Очистка приложения»* используется, чтобы очистить кэш и открыть программу с нуля.
2. Добавляем события *«touch»* для нажатия на кнопки **Продолжить** и **Спасибо, не надо**, чтобы пропускать стандартные вопросы от браузера при первом запуске.  
3. Выбираем поисковую строку и вводим значение *«zennolab»*. Далее нажимаем правой кнопкой мыши по кубику →  
Добавить действие → Android → Эмуляция клавиатуры. В свойства действия пишем текст: *«**AndroidKeys.ENTER**»*.  
4. Нажимаем на ссылку ZennoDroid и открываем ее. 
5. Правой кнопкой мыши нажимаем на кубик с событием, открывающим ссылку → Добавить действие → Логика → Пауза → ставим 10 секунд. После этого добавляем Эмуляцию клавиатуры, которая будет отвечать за сворачивание приложения.

### Как сделать Бэкап (backup)  

> Если есть в этом необходимость, вы можете сохранить состояние браузера в бэкап  
> (куки, авторизации и прочее) и восстановить его перед началом новых действий,  
> чтобы каждый раз не стартовать с чистого браузера.  
>
> *Как правило, это применяется вместе со сменой параметров устройства,  
> об этом читайте подробнее в наших статьях:*  
> 
> - *Настройки устройства (ZD)* (*вставить ссылку позже!*)  
> - *Настройки устройства (Enterprise)* (*вставить ссылку позже!*) 

#### Подробная инструкция:  

1. Нажатие правой кнопкой мыши по полю с кубиками → Добавить действие → Android → Действия с приложениями.  
Далее нужно выбрать действие **Сохранить данные приложения**. В поле **Имя приложения** пишем идентификатор приложения: *«com.android.chrome»*. Также необходимо задать путь для сохранения архива с Бэкапом.
2. Повторяем прошлый пункт, но в действиях выбираем **Восстановить данные приложения**. Указываем путь к архиву  
с нашим Бэкапом.
