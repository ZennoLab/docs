## Описание.  
Данная функция позволяет получить комментарии и ID для всех действий в проекте.  

### Как это работает?  
![Как это работает?](./assets/CopyID/CopyID_pic1.png)  

Достаточно кликнуть по пункту **Копировать ID всех действий** из верхнего меню **Файл**, чтобы все данные автоматически скопировались в буфер обмена. После этого их можно будет вставить в любой текстовый редактор.  
_______________________________________________
## Формат данных.  
Информация о каждом экшене размещается с новой строки. Сначала идёт **комментарий**, затем **символ табуляции**, а потом **ID** экшена.  

Ниже мы приводим пример вывода данных для проекта по названием **`Simple registration.zp`**.  

:::tip **Этот проект поставляется с каждой копией ZennoDroid.**  
Его можно найти на ***Стартовой странице***.
:::
