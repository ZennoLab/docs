### ZennoDroid Lite.
Базовая версия, идеально подходящая для знакомства с платформой и **решения простых задач автоматизации**. Работает исключительно с виртуальными Android-устройствами через **эмуляторы Android MEmu и LDPlayer**.  

Lite версия включает все базовые инструменты для создания сценариев автоматизации, но имеет ограниченный функционал. Работа ведется **в однопоточном режиме**, что подходит для изучения платформы и простых проектов. Лицензию можно **установить только на один компьютер**, хотя это отличный выбор для начинающих пользователей, студентов или тех, кто только изучает возможности автоматизации мобильных приложений.

### ZennoDroid Pro.
Профессиональное решение для серьезных коммерческих проектов. Как и Lite версия, Pro работает с виртуальными устройствами через **эмуляторы MEmu и LDPlayer**. Здесь доступны продвинутые инструменты создания сценариев, поддержка сложных алгоритмов и условной логики, возможности интеграции с внешними сервисами и базами данных.  

Ключевое преимущество Pro версии — **работа в многопоточном режиме без ограничений**, что позволяет одновременно выполнять множество задач и значительно повышает производительность. Лицензия предусматривает **установку сразу на три компьютера**, обеспечивая гибкость в работе.  

:::tip **Одновременный запуск по-прежнему возможен только на одной машине.**
:::
