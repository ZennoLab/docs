# Загрузка данных в проект

Существует несколько способов получения данных для использования в проекте:

- [❗→ Списки](/wiki/spaces/RU/pages/534020137 "/wiki/spaces/RU/pages/534020137") - используются преимущественно для получения построчно данных из текстового документа (например, список e-mail, заголовков для статей и пр.).
- Простые [❗→ Таблицы](/wiki/spaces/RU/pages/534315019 "/wiki/spaces/RU/pages/534315019") и [❗→ Google таблицы](https://zennolab.atlassian.net/wiki/spaces/RU/pages/509411347/Google- "https://zennolab.atlassian.net/wiki/spaces/RU/pages/509411347/Google-") - используются для получения более сложно организованных данных (например, список товаров для интернет-магазина, где построчно представлены разные данные: название, цена, описание и пр.).
- [❗→ Файлы](/wiki/spaces/RU/pages/486309948 "/wiki/spaces/RU/pages/486309948") - когда требуется сразу загрузить в проект всё содержимое файла для дальнейшей работы с ним.
- [❗→ Базы данных](/wiki/spaces/RU/pages/486473806 "/wiki/spaces/RU/pages/486473806") - позволяют хранить большие массивы данных со сложной структурой и связями. Базы данных могут хранится как локально так и удалённо.
- [❗→ Входные настройки](/wiki/spaces/RU/pages/492339208 "/wiki/spaces/RU/pages/492339208") и [❗→ Интерфейс бота (BotUI)](/wiki/spaces/RU/pages/725352539 "/wiki/spaces/RU/pages/725352539")- данные способы используются, когда необходимо задать базовые настройки проекта: указать пути к необходимым файлам, задать режим работы, выбрать капча-сервис и другие необходимые настройки.
