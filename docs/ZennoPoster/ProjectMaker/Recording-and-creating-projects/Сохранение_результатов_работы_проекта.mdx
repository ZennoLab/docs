# Сохранение результатов работы проекта

Если по окончании работы проекта Вам требуется сохранить те или данные, Вы можете использовать те же инструменты, что и для [❗→ загрузки данных](https://zennolab.atlassian.net/wiki/spaces/RU/pages/483426328/1 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/483426328/1"): [❗→ списки](/wiki/spaces/RU/pages/534020137 "/wiki/spaces/RU/pages/534020137"), [❗→ файлы](/wiki/spaces/RU/pages/486309948 "/wiki/spaces/RU/pages/486309948"), [❗→ таблицы](/wiki/spaces/RU/pages/534315019 "/wiki/spaces/RU/pages/534315019"), [❗→ Google таблицы](https://zennolab.atlassian.net/wiki/spaces/RU/pages/509411347/Google- "https://zennolab.atlassian.net/wiki/spaces/RU/pages/509411347/Google-"), [❗→ базы данных](/wiki/spaces/RU/pages/486473806 "/wiki/spaces/RU/pages/486473806").

К примеру, Вы создали шаблон для регистрации на сайте. Использовали при этом переменные профиля для логина и пароля: `{ -Profile.Login- }` и `{ -Profile.Password- }`.

Для сохранения этих данных Вы используете один из экшенов: [❗→ Операции со списком -&gt; Добавить строку](https://zennolab.atlassian.net/wiki/spaces/RU/pages/534085798#%D0%94%D0%BE%D0%B1%D0%B0%D0%B2%D0%B8%D1%82%D1%8C-%D1%81%D1%82%D1%80%D0%BE%D0%BA%D1%83 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/534085798#%D0%94%D0%BE%D0%B1%D0%B0%D0%B2%D0%B8%D1%82%D1%8C-%D1%81%D1%82%D1%80%D0%BE%D0%BA%D1%83"), [❗→ Операции с таблицами -&gt; Добавить строку](https://zennolab.atlassian.net/wiki/spaces/RU/pages/534052972#%D0%94%D0%BE%D0%B1%D0%B0%D0%B2%D0%B8%D1%82%D1%8C-%D1%81%D1%82%D1%80%D0%BE%D0%BA%D1%83 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/534052972#%D0%94%D0%BE%D0%B1%D0%B0%D0%B2%D0%B8%D1%82%D1%8C-%D1%81%D1%82%D1%80%D0%BE%D0%BA%D1%83") или [❗→ Файлы -&gt; Записать](https://zennolab.atlassian.net/wiki/spaces/RU/pages/486309948#%D0%97%D0%B0%D0%BF%D0%B8%D1%81%D0%B0%D1%82%D1%8C-%D1%82%D0%B5%D0%BA%D1%81%D1%82 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/486309948#%D0%97%D0%B0%D0%BF%D0%B8%D1%81%D0%B0%D1%82%D1%8C-%D1%82%D0%B5%D0%BA%D1%81%D1%82"). И в поле «значение» добавляете эти переменные с нужным Вам разделителем.

Если например, проект у Вас по той или иной причине выполняется с ошибкой (такое бывает часто при использовании некачественных прокси), Вы можете сохранить необходимые данные, которые были взяты во время работы проекта (например e-mail из списка), используя [❗→ Bad end](https://zennolab.atlassian.net/wiki/spaces/RU/pages/534020265/Bad+End "https://zennolab.atlassian.net/wiki/spaces/RU/pages/534020265/Bad+End").
