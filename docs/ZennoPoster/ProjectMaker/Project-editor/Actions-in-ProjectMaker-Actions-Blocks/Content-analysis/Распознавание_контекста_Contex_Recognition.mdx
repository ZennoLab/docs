## Описание

Позволяет определить тематику контента.

:::info Информация
Работа возможна только с англоязычным контентом
:::

![image-20200816-173257](./assets/Contex_Recognition/Распознавание_контекста_(Contex_Recognition)_pic1.png)



## Как добавить действие в проект?

Через контекстное меню **Добавить действие** → **Анализ содержимого** → **Context Recognition**

![image-20200816-173551](./assets/Contex_Recognition/Распознавание_контекста_(Contex_Recognition)_pic2.png)



Либо воспользуйтесь [❗→ умным поиском](https://zennolab.atlassian.net/wiki/spaces/RU/pages/506200090/ProjectMaker+7#%D0%A3%D0%BC%D0%BD%D1%8B%D0%B9-%D0%BF%D0%BE%D0%B8%D1%81%D0%BA-%D0%B4%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%B8%D0%B9 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/506200090/ProjectMaker+7#%D0%A3%D0%BC%D0%BD%D1%8B%D0%B9-%D0%BF%D0%BE%D0%B8%D1%81%D0%BA-%D0%B4%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%B8%D0%B9").

## Для чего это используется?

- Определение тематики текста
- Фильтрация сайтов по тематике для размещения ссылок
- Парсинг контента
- Определение текста по заданным критериям

## Как работать с экшеном?

Если вам нужно определить тематику текста вне проекта, то воспользуйтесь инструментом [❗→ Распознавание контекста](/wiki/spaces/RU/pages/534315397 "/wiki/spaces/RU/pages/534315397").

![image-20200816-173954](./assets/Contex_Recognition/Распознавание_контекста_(Contex_Recognition)_pic3.png)



1. Поле для ввода текста или переменной.
2. Тематики : 
a) **Общие** - около 20 тематик общего содержания.
b) **Подробные** - более узкие направления текста, около 250.
3. Задать максимальное количество тематик, которые должен выдать анализатор, можно использовать переменную.
4. Минимальная релевантность текста от 0 до 100%, можно воспользоваться переменной.
5. Переменная для результата.

:::info Информация
Все тематики, попадающие под критерии, будут помещены в переменную с разделителем “;”
:::

Спарсить текст с главной страницы сайта и определить минимум три тематики.

Она находится в экшене «[❗→ Выделение основной статьи](https://zennolab.atlassian.net/wiki/spaces/RU/pages/534315067/ "https://zennolab.atlassian.net/wiki/spaces/RU/pages/534315067/")».

## Пример использования

Contex Recognition пользуется популярностью среди SEO-специалистов. Одно из самых частых применений - это определение тематики сайтов для размещения ссылок.

Вместо того, чтобы публиковать свои ссылки на всех ресурсах подряд и рисковать получить бан от поисковых систем, Вы сначала можете определить тему текста страницы, на которой хотите оставить ссылку. Если текст не подходит под тему вашего сайта, то не стоит оставлять там ссылки. Аналогично вы можете проверить тематичность всех исходящих ссылок на своём сайте.

Допустим, у вас есть база ссылок для постинга. С помощью **Context Recognizer** вы можете разбить вашу базу на несколько баз по контексту. Потом, когда вам нужно будет прорекламировать сайт, вы возьмете не полную базу, а ту, которая соответствует тематике сайта. Вы сможете постить статью про страхование машин в блог с автомобильной тематикой, а не в блог, где публикуют анонсы новых фильмов.

Можно пропарсить сайт (например, блог) и найти страницы, которые лучше всего подходят по тематике к вашей рекламе. Оставляя релевантные комментарии и посты, вы получите не только тематичные ссылки, но больший шанс пройти модерацию, что очень важно на качественных ресурсах.
