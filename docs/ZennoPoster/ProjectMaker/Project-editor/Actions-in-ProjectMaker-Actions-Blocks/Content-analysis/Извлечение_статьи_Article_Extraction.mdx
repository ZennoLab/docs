## Описание

Позволяет получить основную статью на странице ресурса.

![image-20200816-163409](./assets/Извлечение_статьи_(Article_Extraction)/Извлечение_статьи_(Article_Extraction)_pic1.png)



## Где можно применить: 

- Парсинг контента с ресурсов
- Работа к текстом

## Как добавить действие в проект?

Через контекстное меню **Добавить действие** → **Анализ** **содержимого**→ **Article Extraction**

## Как работать с экшеном?

![image-20200816-164002](./assets/Извлечение_статьи_(Article_Extraction)/Извлечение_статьи_(Article_Extraction)_pic3.png)



1. Вкладка с загруженной страницей : 
a) **Активная** - таб, который у вас в данный момент перед глазами.
b) **Первая** - первое окно слева.
c) **По имени** - указать имя таба или переменную учитывая регистр букв.
d) **По номеру** - задаём номер вкладки. Нумерация идёт слева направо начиная с 0. Если необходимо закрыть самую первую вкладку, то в поле указываем ноль , последующие вкладки идут по счету 1, 2, 3….
2. Переменная для результата.

## Пример использования: 

Перейти на страницу и получить текст. Полученный контент складываем в список для дальнейшей работы.

![image-20200816-165140](./assets/Извлечение_статьи_(Article_Extraction)/Извлечение_статьи_(Article_Extraction)_pic6.png)



1. Переходим на страницу.
2. Получаем основную статью и кладём в переменную.
3. Записываем в список.

Позволит максимально быстро парсить текст, не прибегая к использованию множества инструментов.
