## Описание

Таб браузера - это вкладка с сайтом в браузере

![image-20200806-203005](./assets/Управление_вкладкой/Управление_вкладкой_браузера_(табом_браузера)_pic1.png)



На картинке у нас открыт таб с именем “Яндекс” и нём загружена стартовая страница Яндекса.

## Где можно применить: 

- Работа с несколькими сайтами одновременно
- Закрытие всплывающих сайтов
- Открыть дополнительное окно браузера с нужным сайтом
- Перейти к нужному окну с сайтом
- Присвоение имени окну с сайтом для удобной навигации при работе

## Как добавить действие в проект?

Через контекстное меню **Добавить действие** → **Браузер** → **Управление табом**

## Как с этим работать?

### Активировать вкладку

![image-20200806-203422](./assets/Управление_вкладкой/Управление_вкладкой_браузера_(табом_браузера)_pic3.png)



:::info Информация
Данное действие работает только с уже открытыми окнами.
:::

- **По имени** - в браузере будет активно окно с именем, указанным в действии. Можно указать переменную с названием вкладки.

:::info Информация
Важно соблюдать регистр имени таба. Если у вас вкладка называется Google, а в экшене имя google, то действие не будет выполнено.
:::

- **По номеру** - откроется вкладка по номеру начиная слева направо.

![image-20200806-204328](./assets/Управление_вкладкой/Управление_вкладкой_браузера_(табом_браузера)_pic4.png)



:::info Информация
Нумерация вкладок начинается с 0. Первая вкладка всегда будет стоять под номером 0, все последующие соответственно 1 ,2 ,3
:::

  

### Открыть вкладку

- Кубик создаст новое окно браузера с заданным именем. Так же в поле можно указать переменную если вы генерируете название случайным образом или используете заготовленный список.

![image-20200806-205259](./assets/Управление_вкладкой/Управление_вкладкой_браузера_(табом_браузера)_pic5.png)



  

### Закрыть вкладку

![image-20200806-204504](./assets/Управление_вкладкой/Управление_вкладкой_браузера_(табом_браузера)_pic6.png)



- *Активную - вкладка , которая у вас перед глазами.
- *По имени - указываем имя или переменную , содержащую имя вкладки соблюдая регистр названия.
- *По номеру - указываем номер вкладки. Нумерация идёт слева направо начиная с 0. Если необходимо закрыть самую первую вкладку, то в поле указываем ноль , последующие вкладки идут по счету 1, 2, 3….

## Пример использования: 

Вы регистрируетесь на сайте и нужно подтвердить учетную запись через почту.

![image-20200816-183125](./assets/Управление_вкладкой/Управление_вкладкой_браузера_(табом_браузера)_pic7.png)



1. Выбираем открыть вкладку указав имя
2. Загружаем сайт и производим нужные действия
3. Выбираем закрыть вкладку указав имя окна из пункта 1 или номер
