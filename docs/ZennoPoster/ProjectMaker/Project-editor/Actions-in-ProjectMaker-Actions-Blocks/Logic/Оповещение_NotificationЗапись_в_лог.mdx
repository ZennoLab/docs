# Оповещение (Notification/Запись в лог)

  

## Описание

Данный кубик позволяет Вам оповещать пользователя шаблона о событиях и действиях, происходящих в проекте. Сообщения будут отображаться в [❗→ Окне лога](https://zennolab.atlassian.net/wiki/spaces/RU/pages/725352532 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/725352532").

  

## Как добавить действие в проект?

Через контекстное меню **Добавить действие** → **Логика** → **Оповещение**

![image-20210606-153304](./assets/Оповещение_(NotificationЗапись_в_лог)/Оповещение_(NotificationЗапись_в_лог)_pic1.png)



Либо воспользуйтесь [❗→ умным поиском](https://zennolab.atlassian.net/wiki/spaces/RU/pages/506200090/ProjectMaker+7#%D0%A3%D0%BC%D0%BD%D1%8B%D0%B9-%D0%BF%D0%BE%D0%B8%D1%81%D0%BA-%D0%B4%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%B8%D0%B9 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/506200090/ProjectMaker+7#%D0%A3%D0%BC%D0%BD%D1%8B%D0%B9-%D0%BF%D0%BE%D0%B8%D1%81%D0%BA-%D0%B4%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%B8%D0%B9").

  

## Где это можно применить?

- Логирование действий в шаблоне
- Оповещение пользователе об изменениях
- Оповещение о версии шаблона

  

## Как работать с экшеном?

![image-20210606-154408](./assets/Оповещение_(NotificationЗапись_в_лог)/Оповещение_(NotificationЗапись_в_лог)_pic2.png)



### Текст сообщения

Сюда указывается текст, который отобразится у пользователя. Возможно использование любых макросов. 

### Уровень сообщения

Разница лишь в иконках, которые отображаются перед сообщением.

![image-20210606-155203](./assets/Оповещение_(NotificationЗапись_в_лог)/Оповещение_(NotificationЗапись_в_лог)_pic3.png)



:::note На заметку
В Окне лога можно сортировать сообщения по типу.
:::

### Цвет сообщения

:::info Информация
Добавлено в ZennoPoster 7.2.1.0
:::

![image-20210606-155316](./assets/Оповещение_(NotificationЗапись_в_лог)/Оповещение_(NotificationЗапись_в_лог)_pic4.png)



Позволяет выбрать цвет для фона сообщения в [❗→ окне лога](https://zennolab.atlassian.net/wiki/spaces/RU/pages/725352532 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/725352532").

![image-20210606-155506](./assets/Оповещение_(NotificationЗапись_в_лог)/Оповещение_(NotificationЗапись_в_лог)_pic5.png)



:::note На заметку
В Окне лога можно сортировать сообщения по их цвету.
:::

### Показывать в ZennoPoster

Если установлено - оповещение будет отображаться как в ProjectMaker, так и в ZennoPoster

Если не установлено - только в PM.

### Выводить только в лог

Если установлено - оповещение будет выводиться только в лог, но не будет всплывать на рабочем столе.

![image-20200811-164737](./assets/Оповещение_(NotificationЗапись_в_лог)/Оповещение_(NotificationЗапись_в_лог)_pic6.png)



:::note На заметку
В настройках программы можно включить опцию Выводить оповещение только в лог и тогда все сообщения будут ВСЕГДА выводится ТОЛЬКО в лог.
:::

### Автоматически закрывать через N секунд

Опция, которая отвечает за время закрытия оповещения при всплывании на рабочем столе.

  

## Пример использования

Например, вы можете оповестить пользователя об успешном выполнения какого-либо действия, этапах работы шаблоны, количестве обработанных данных и т.п.
