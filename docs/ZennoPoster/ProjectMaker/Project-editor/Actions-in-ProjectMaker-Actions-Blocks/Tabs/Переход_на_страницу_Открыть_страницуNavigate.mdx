## Описание

Загрузит нужный сайт или точную страницу в указанной вкладке браузера.

| ![В активной вкладке браузера открылась точно заданная страница сайта](./assets/Переход_на_страницу_(Открыть_страницуNavigate)/Переход_на_страницу_(Открыть_страницуNavigate)_pic1.png) |
| :--: |
| В активной вкладке браузера открылась точно заданная страница сайта |



## Где можно применить: 

- Загрузка сайта
- Открытие точной страницы сайта
- Активация учетных записей
- Имитация перехода с сайта на нужный ресурс
- Загрузка сайта в неактивной вкладке браузера

## Как добавить действие в проект?

## Как работать с экшеном?

- **URL** - в данном поле указываем страницу сайта или переменную, содержащую адрес для последующего открытия.

![url_ph](./assets/Переход_на_страницу_(Открыть_страницуNavigate)/Переход_на_страницу_(Открыть_страницуNavigate)_pic3.jpg)



- **Referrer -** здесь указываем ресурс **с** которого был произведён переход, тем самым показывая сайту, что мы живой пользователь. Можно вставить переменную с готовым списком.

![реферрер_ph](./assets/Переход_на_страницу_(Открыть_страницуNavigate)/Переход_на_страницу_(Открыть_страницуNavigate)_pic4.jpg)



Мы перешли на сайт Яндекс.Почта со стартовой странице Яндекса
- **Какая вкладка** - указываем точно в каком окне загружать сайт или страницу.

1. *Активная - вкладка, которая находится непосредственно у нас перед глазами.
2. *Первая - загрузит сайт в самой крайнем окне слева - в первом, отсчет слева.
3. *По имени - указываем имя вкладки или переменную с именем с учетом регистра букв.
4. *По номеру - номер вкладки слева направо начиная с нуля, первая окно всегда имеет значение 0, последующие вкладки соответственно 1, 2, 3 и тд.

:::info Информация
Если указываем загрузить страницу в первой, по имени или по номеру вкладке, то активное в данный момент окно не изменится и не активируется. Загрузка произойдёт в неактивном табе.
:::

![какая вкладка](./assets/Переход_на_страницу_(Открыть_страницуNavigate)/Переход_на_страницу_(Открыть_страницуNavigate)_pic5.jpg)


## Пример использования: 

Необходимо во второй вкладке загрузить сайт почты для активации учетной записи

![image-20200816-183755](./assets/Переход_на_страницу_(Открыть_страницуNavigate)/Переход_на_страницу_(Открыть_страницуNavigate)_pic6.png)



1. В активном окне делаем необходимые действия для регистрации
2. Указываем вкладку с номером 1(помним, что окна считаются с нуля) загружаем параллельно сайт почты
3. Переходим в нужный таб и активируем аккаунт
