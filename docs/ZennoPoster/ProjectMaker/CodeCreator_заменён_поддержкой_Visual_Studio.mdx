# 📑 CodeCreator (заменён поддержкой Visual Studio)

:::warning Внимание
Внимание! CodeCreator не поддерживается начиная с ZennoPoster 7 и заменён на Visual Studio.Для использования программы - используйте ZennoPoster 5.
:::

**CodeCreator**- это по функциональности тоже самое, что и ProjectMaker, только в CodeCreator записывается, редактируется и отлаживается код, написанный на C# или PHP, а не проекты, как в ProjectMaker.

CodeCreator**доступен всем пользователям через меню Пуск или в основной папке программы.

Видеоинструкции по ссылкам на YouTube [здесь](https://www.youtube.com/watch?v=k_DRWhsBcx8 "https://www.youtube.com/watch?v=k_DRWhsBcx8") и [здесь](https://www.youtube.com/watch?v=tpF3bKE2pFA "https://www.youtube.com/watch?v=tpF3bKE2pFA").
