# Профиль (PM)

![image-20240112-081209.png](./assets/Профиль_(PM)/Профиль_(PM)_pic1.png)



Позволяет указать настройки профиля, который будет генерироваться по умолчанию при старте проекта.

## Настройки заполнения профиля

### **Национальность по умолчанию**

Национальность для генерации профиля. Её можно изменить и на уровне шаблона, в настройках [❗→ профиля проекта](https://zennolab.atlassian.net/wiki/spaces/RU/pages/483426308 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/483426308").

![image-20200812-172201](./assets/Профиль_(PM)/Профиль_(PM)_pic2.png)



### **E-mail для регистраций**

Указанный в этой настройке E-mail будет использоваться для всех новых профилей.

Также, можно изменить через переназначение полей профиля в [❗→ операциях над профилем](https://zennolab.atlassian.net/wiki/spaces/RU/pages/486539291 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/486539291"). Значение хранится в [❗→ переменной](https://zennolab.atlassian.net/wiki/spaces/RU/pages/735608872 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/735608872") окружения `{ -Profile.Email- }`.

### **Пароль от e-mail для регистраций**

Указанный в этой настройке пароль от e-mail будет использоваться для всех новых профилей.

Также, можно изменить через переназначение полей профиля в [❗→ операциях над профилем](https://zennolab.atlassian.net/wiki/spaces/RU/pages/486539291 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/486539291"). Значение хранится в [❗→ переменной](https://zennolab.atlassian.net/wiki/spaces/RU/pages/735608872 "https://zennolab.atlassian.net/wiki/spaces/RU/pages/735608872") окружения `{ -Profile.EmailPassword- }`.
