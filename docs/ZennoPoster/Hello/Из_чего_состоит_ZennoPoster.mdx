# Из чего состоит ZennoPoster

ZennoPoster - это программный пакет, который состоит из нескольких приложений, позволяющих автоматизировать Ваши действия в веб-браузере. Данное программное обеспечение подойдет для каждого пользователя - как для SEO-шника, занимающегося продвижением в сети, так и для опытного веб-программиста, кодирующего сложные боты.

Пакет ZennoPoster включает в себя следующие приложения:

**ZennoPoster** - основное приложение, предназначенное для выполнения автоматизированных шаблонов действий (ботов) в браузере. Данное приложение позволяет управлять потоками при выполнении шаблонов, использовать различные прокси для Ваших ботов, а также имеет [❗→ встроенный планировщик](/wiki/spaces/RU/pages/534086320 "/wiki/spaces/RU/pages/534086320") для выполнения шаблонов по расписанию.

**ProjectMaker** - приложение для создания автоматизированных шаблонов действий. Именно в нем Вы будете записывать и автоматизировать свои действия на веб-страницах. Сделать это достаточно легко - программа имеет набор [❗→ стандартных действий (экшенов)](https://zennolab.atlassian.net/wiki/spaces/RU/pages/486342706/ProjectMaker "https://zennolab.atlassian.net/wiki/spaces/RU/pages/486342706/ProjectMaker") для веб-браузера, из блоков которых конструируется шаблон, а также режим записи действий, который записывает в шаблон все Ваши действия в браузере. Приложение также позволяет работать со [❗→ списками](/wiki/spaces/RU/pages/534053375 "/wiki/spaces/RU/pages/534053375"), [❗→ таблицами](/wiki/spaces/RU/pages/735903776 "/wiki/spaces/RU/pages/735903776"), [❗→ Google Таблицами](https://zennolab.atlassian.net/wiki/spaces/RU/pages/509411347/Google- "https://zennolab.atlassian.net/wiki/spaces/RU/pages/509411347/Google-"), [❗→ базами данных](/wiki/spaces/RU/pages/486473806 "/wiki/spaces/RU/pages/486473806"), текстовыми и другими [❗→ файлами](/wiki/spaces/RU/pages/486309948 "/wiki/spaces/RU/pages/486309948") на Вашем компьютере и подгружать их в веб, значительно упрощая работу SEO-специалистов.
