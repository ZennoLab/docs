# ZennoPoster - Demo

## Ограничения Демоверсии

1. Программа работает 14 дней
2. Доступен только один поток выполнения
3. Использование прокси отключено
4. Каптчи вводятся только вручную (нельзя использовать сервисы типа [CapMonster.cloud](https://capmonster.cloud/ "https://capmonster.cloud/"))
5. Если вы не ввели свой email при установке демоверсии, проекты нельзя передавать другим пользователям: то есть по окончании срока демоверсии проекты станут неработоспособными.
6. Демо версия не является заменой Lite лицензии: при использовании демо вы можете заниматься тестированием функций, написанием шаблонов, при этом отладка (выполнение шаблона в ProjectMaker) и выполнение шаблонов в ZennoPoster/ZennoDroid запускает счетчик, отсчитывающий время работы автоматизации. По достижении этим счетчиком значения в 10 минут пользователю предлагается ввести вручную каптчу для дальнейшего продолжения работы.
7. Нельзя использовать [❗→ bat файлы для запуска проектов](/wiki/spaces/RU/pages/1940979747 "/wiki/spaces/RU/pages/1940979747").

Если проекты, сделанные в демо версии перестали открываться в новой демке, или в лицензионной версии - можете прислать их на support@zennolab.com с email лицензии, мы поможем восстановить доступ к проектам для вашей текущей лицензии.

## Как продлить срок demo

Если срок демо лицензии истек, вы сможете попробовать снова через 16 дней.
