## **Запуск и настройка**


**ZennoPoster7** открывает новые горизонты автоматизации благодаря интеграции с **ZennoBrowser** — профессиональным антидетект-браузером.

### **Ключевые преимущества интеграции**

**Основные возможности:**

 - Управление профилями в режиме реального времени;
 - Работа с прокси на продвинутом уровне;
 - Изолированные среды для каждого процесса;
 - Контроль цифровых отпечатков.

**Основные функции:**

 - Создание профилей с уникальными настройками;
 - Запуск браузерных инстансов и управление ими; 
 - Удаление профилей без потери данных.


**Процесс интеграции обеспечивает:**

 - Бесшовное взаимодействие между продуктами;
 - Сохранение интерфейса ZennoPoster7;
 - Расширение возможностей за счет функций ZennoBrowser.

### **Какие возможности даёт интеграция?**

#### **1. Получение профилей ZennoBrowser**

Действие "Получить профили ZennoBrowser" предназначено для извлечения информации о существующих профилях браузера из системы ZennoBrowser.

![launch_and_settings4](..\zp_integr\assets\launch_and_settings\launch_and_settings4.png)

**Принцип работы** 

Действие выполняет запрос к ZennoBrowser и возвращает полную информацию о всех доступных профилях в виде структурированных данных.

**Формат результата**

Результат выполнения представляет собой JSON-массив объектов, где каждый объект содержит детальную информацию об отдельном профиле:
 - ID профиля;
 - Имя профиля;
 - Настройки браузера; 
 - Статус профиля;
 - Данные прокси;
 - Другие метаданные.

#### **2. Обработка результатов**

Для корректной работы с полученными данными рекомендуется использовать действие "Парсинг JSON", которое позволит:
 - Извлечь конкретные параметры профилей;
 - Отфильтровать профили по заданным критериям;
 - Преобразовать данные в удобный для работы формат.

#### **3. Обработка ошибок**

В случае проблем с подключением к ZennoBrowser (недоступность сервиса, неверные настройки соединения) действие вернёт соответствующую ошибку, которую необходимо обработать в логике проекта.

#### **4. Сохранение результата**

Укажите переменную для сохранения JSON-ответа в поле "Поместить в переменную" для дальнейшего использования в проекте.
