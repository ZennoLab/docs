## **Интеграция с действием "Запустить инстанс"**

Это действие предназначено для запуска браузера с использованием конкретного профиля ZennoBrowser по его идентификатору.

![integr_instans1](..\zp_integr\assets\integr_instans\integr_instans1.png)
![integr_instans2](..\zp_integr\assets\integr_instans\integr_instans2.png)

### **Принцип работы**

Действие принимает ID профиля ZennoBrowser и инициирует запуск браузерного инстанса с соответствующими настройками профиля, включая:
 - Цифровые отпечатки браузера;
 - Настройки прокси;
 - Пользовательские данные;
 - Расширения и конфигурации.

### **Настройка профиля**

При первом открытии настроек действия автоматически загружается актуальный список профилей из ZennoBrowser. Доступны два способа указания профиля:

 1. **Ручной ввод ID:** введите идентификатор профиля в соответствующее поле.

 2. **Выбор из списка:**

 -> Сначала выберите папку профилей из выпадающего списка.

 -> Затем выберите конкретный профиль внутри выбранной папки.

### **Обновление списка профилей**

Для получения актуальной информации о доступных профилях используйте кнопку "Обновить" в настройках действия. Это особенно полезно при добавлении новых профилей в ZennoBrowser.

![integr_instans3](..\zp_integr\assets\integr_instans\integr_instans3.png)

### **Новая опция браузера**

В стандартное действие "Запустить инстанс" добавлен новый тип браузера — "Chromium from ZennoBrowser (beta)".

![integr_instans4](..\zp_integr\assets\integr_instans\integr_instans4.png)

Эта интеграция обеспечивает единообразный подход к запуску различных типов браузеров в рамках одного действия.
