## **Создание профилей**

### **Способы создания**

Кнопка "Новый профиль" открывает окно с двумя режимами:

![creating_profiles1](..\main_feautures\assets\creating_profiles\creating_profiles1.png)

#### **Единичное создание профиля в режиме “Новый профиль”**

**Возможности:** 

 - Задание имени профиля;
 - Импорт cookie-файлов;
 - Добавление заметок;
 - Задание расширенных параметров.

К **расширенным параметрам** относятся экран, ОЗУ, язык, геолокация и прочие, настройка которых напрямую влияет на успех выполнения задач и сохранение анонимности профиля:

![creating_profiles2](..\main_feautures\assets\creating_profiles\creating_profiles2.png)
![creating_profiles3](..\main_feautures\assets\creating_profiles\creating_profiles3.png)

#### **Массовое создание профилей в режиме “Массовое создание”**

**Возможности:** 
 - До 1000 профилей за одну операцию;
 - Настройка уникального назначения прокси.

**Общие настройки для всех типов:**
 - Выбор папки назначения;
 - Тип профиля — облачный или локальный;

### **Настройка тегов**

#### **Настройки прокси при создании:**

| **Опция** | **Описание** | 
|-------------|-------------|
| Без прокси | Создание профиля без прокси-сервера |
| Новый прокси | Автоматическое назначение нового прокси |
| Выбор прокси | Ручной выбор конкретного прокси |
| Список прокси | Для массового создания с выбором из списка |
