## **Структура и основные компоненты**

**ZennoBrowser** включает в себя два основных модуля для эффективного управления:

 1. Менеджер профилей — централизованное управление локальными и облачными профилями.
 2. Менеджер прокси — управление прокси-серверами и их настройка.

### **Управление профилями**

Организация и структура хранения:
 - Локальные профили сохраняются на вашем устройстве;
 - Облачные профили синхронизируются через облако;
 - Папки — создаётся иерархическая структура для удобной навигации.

**Интерфейс таблицы профилей:**

![19](..\basics\assets\Main_Features\19.png)

**Настройки отображения:**
 - Редактируемая таблица с возможностью скрытия столбцов;
 - Перестановка столбцов в удобном порядке;
 - До 1000 профилей на одной странице.

**Столбцы таблицы:**

| **Столбец** | **Описание** | 
|-------------|-------------|
| Имя профиля | Пользовательское название для идентификации | 
| Управление | Элементы управления профилем | 
| Теги | Метки для категоризации и быстрого поиска |
| Прокси| Назначенные прокси-серверы | 
| Заметки | Дополнительные комментарии |
| Время работы | Продолжительность использования | 
| Последний запуск | Дата последней активности |


**Поиск и сортировка:**

 - Сортировка — по любому столбцу таблицы;
 - Поиск — профилей по имени, заметкам и ID.

### **Массовые операции**

**Выбор профилей:** система чекбоксов позволяет выбирать несколько профилей для групповых действий.

**Доступные массовые действия:**
 - Запуск профилей;
 - Остановка профилей;
 - Замена прокси;
 - Перемещение в другую папку;
 - Управление тегами;
 - Экспорт cookie-файлов;
 - Массовая смена IP мобильных прокси;
 - Удаление профилей.

## **Редактирование профилей**

Каждый профиль поддерживает индивидуальное редактирование параметров отпечатка браузера.

*Что такое отпечаток браузера (Browser Fingerprint )?*

**Отпечаток браузера** — это как уникальный паспорт вашего компьютера или телефона в интернете: набор технических данных о вашем устройстве и браузере, который позволяет сайтам идентифицировать вас даже без использования cookies. Управление отпечатком браузера — критически важный аспект обеспечения цифровой приватности и эффективного мультиаккаунтинга.

*Как это работает?*

Сайт собирает информацию автоматически при его посещении. **Система анализирует:**
 - Настройки браузера;
 - Технические характеристики устройства;
 - Конфигурацию сети;
 - Установленные плагины;
 - Другие параметры системы.

**Основные компоненты отпечатка:**

 - Версия браузера;
 - Разрешение экрана;
 - Установленные шрифты;
 - Системные настройки;
 - Сетевые параметры.

Уникальность отпечатка обеспечивается **комбинацией** всех этих параметров.

## **Менеджер прокси**

**Менеджер прокси** предоставляет централизованное управление прокси с возможностью:
 - Создания списков;
 - Проверки работоспособности;
 - Массовых операций.

**Структура интерфейса:**

![24](..\basics\assets\Main_Features\24.png)

**Таблица прокси** содержит следующие столбцы:

| **Столбец** | **Описание** | 
|-------------|-------------|
| Имя прокси | Пользовательское название для идентификации |
| Тип | Протокол подключения (HTTP, SOCKS5, и др.) |
| Хост:порт | Адрес и порт прокси-сервера |
| Логин:пароль | Данные для аутентификации |


**Организация:**
 - Списки прокси — создание тематических групп;
 - Отображение — до 1000 прокси на одной странице;
 - Счётчик — показывает количество добавленных прокси.

## **Добавление прокси**

### **Поддерживаемые форматы**

**Базовые форматы:**

192.168.0.1:8000

http://192.168.0.1:8000

192.168.0.1:8000:login:password

socks5://login:password@192.168.0.1:8000


**IPv6 формат:**

http://[2a06:c006:bd1e:00a1:2d4c:1cbd:4653:e7bc]:8000


### **Правила добавления**

**Ограничения:** 
 - Прокси должны разделяться переносом строки;
 - Максимум 1000 прокси за одно добавление (вставка в текстовое поле).

**Дополнительные возможности:** 
 - Именование — добавьте :name в конце строки, например, так: 
   socks5://login:password@192.168.0.1:8000:MySuperProxy0001
 - Тип по умолчанию — если не указан, используется HTTP;
 - IPv6 поддержка — заключайте адреса в квадратные скобки [ ].

### **Массовые операции**

**Выбор прокси:**
 Используйте чекбоксы для выбора нескольких прокси одновременно.

**Доступные действия:** 
 1. Проверка прокси — тестирование работоспособности выбранных серверов.
 2. Удаление — массовое удаление неработающих или ненужных прокси.

### **Проверка работоспособности**

**Функция проверки:** 
 - Кнопка "Проверить прокси" для тестирования соединения;
 - Обновление статуса в реальном времени;
 - Проверка отдельных прокси или целых списков.

### **Управление списками**

**Создание списков:**
 - Организация прокси по назначению или регионам;
 - Выбор нужного списка из выпадающего меню;
 - Быстрое переключение между списками.
