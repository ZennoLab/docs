## **Начало работы**

С ZennoBrowser можно работать в двух режимах: ручная работа с профилями и интегрированный в связке с ZennoPoster 7.

**Ручная работа с профилями:**

 - Ручная работа с антидетект-браузером;
 - Локальное управление профилями;
 - Оптимален для небольших задач.

**Интегрированный режим:**

 - Автоматизация процессов через ZennoPoster;
 - Масштабные задачи с использованием алгоритмов;
 - Комплексное решение для объёмных проектов.

### **Преимущества каждого режима**

**Ручная работа с профилями:**

 - Гибкость в управлении;
 - Простота настройки;
 - Мгновенный старт работы.

**Интегрированный режим:**

 - Автоматизация рутинных задач;
 - Масштабируемость процессов;
 - Эффективность при работе с большими объёмами.


Выбирайте **ручную работу** с профилями, если:

 - Работаете с небольшим количеством профилей;
 - Предпочитаете ручное управление;
 - Решаете точечные задачи.

Используйте **интеграцию**, если:

 - Нужны автоматизированные процессы;
 - Работаете с масштабными проектами;
 - Требуется обработка большого объёма данных.
