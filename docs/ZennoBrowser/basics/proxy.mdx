## **Настройка прокси**


 1. Откройте Менеджер прокси.
 2. Нажмите кнопку добавления прокси.
 3. Введите прокси в одном из поддерживаемых форматов:

192.168.0.1:8000

http://192.168.0.1:8000

192.168.0.1:8000:login:password

socks5://login:password@192.168.0.1:8000

![proxy1](..\basics\assets\proxy\proxy1.png)


 4. Нажмите "Проверить прокси" для тестирования работоспособности.
 5. Создайте список прокси для удобной организации.

![proxy2](..\basics\assets\proxy\proxy2.png)

### Правила добавления прокси:

 - Один прокси в одну строку;
 - Максимум 1000 прокси за операцию;
 - По умолчанию используется протокол HTTP;
 - Для IPv6 заключайте адреса в квадратные скобки [ ].
